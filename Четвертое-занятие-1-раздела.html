Работа с cross dj

Микширование аудио-дорожек. Знакомство с интерфейсом.

Cross DJ 4 - приложение для микширования аудиодорожек. С помощью нее можно создавать качественную музыку, применять звуковые фитры к собственным композициям, а также просто воспроизводить музыку. Она находится в свободном доступе для установки на ПК операционных систем iOs, Windows, Android и macOs.​ Для начала работы с приложением перейдите на сайт. Доступна бесплтаная демо версия на 14 дней. 

https://www.mixvibes.com/cross-dj-4/

На главной странице сайта нажмите кнопку "DEMO".

Микширование - это сведение звуковых дорожек двух и более звуковых файлов в одну звуковую дорожку. Для начала соединения двух д​орожек загрузите в приложение файлы. Например, можно использовать треки, уже имеющиеся в программе. Для этого в меню "Tracks" выберите подходящие, нажмите по ним левой кнопкой мыши и выберите пункт "Load In Player A" первого трека, и "Load In Player B" второго трека соотвественно.


Воспроизведите первый трек и поочередно примените фильтры "Filter Delay", "Ground Phaser", "Bit Crash" к нему поочередно, настроив силу эффекта и нажав кно​пку "On". Правее расположены регуляторы низких, с​редних и высоких частот. Также примените их и отследите изменение звучание музыкального трека.  Также настройте громкость канала, изменив положение рычага со иконкой наушников. Инструмент "Gain" предназнчаен для настройки уровня громкости звучания в общем канале при проигрывании треков из дух каналов одновременно. Вернуть начальные значения можно с помощью двойного щелчка по инструменту.

Перед тем, как записать сведение двух треков заранее выберите музыкальные фильры "Woosh" для первого трека и "Metalizer" для второго. Поочередно включайте каждый последующий фильтр "LO", "MID", "HI". После того, как каждый из них будет применен примерно по 5 сек, примените инструмент громкости, снизив ее до минимума, и перейдите к проигрыванию второго трека и применению фильтров к нему. Для начала записи нажмите кнопку "REC" в верхнем меню. Нажмите кнопку, чтобы открыть видео-инструкцию, содержащую рекомендации по микшированию.



ВИДЕО

Чтобы увидеть результат, передите в пункт "All Recordings" нижнего меню. Прослушайте полученный трек. Все записи хранятся в папке приложения на вашем жестком диске ПК.
