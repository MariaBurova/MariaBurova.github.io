Soundtrap.com

Совместное создание музыкальных композиций в режиме онлайн

В первом занятии раздела уже описывались отличительные харакетристики данного онлайн аудиоредактора. Приступите к работе, пройдя регистрацию с помощью Google аккаунта.  Далее нажмите на кнопку "Enter Studio".

Далее выберите пункт "Music". "Demos" содержит жанровые проекты, можете ознакомиться и выделить для себя полезные моменты.

Для риглашения пользователя для совместной работы нажмите на значок в правом верхнем углу экрана "Collaborators", далее выберите "Invite". Приглашенный пользователь должен быть зрегистрован в приложении. Для поиска введите его email адрес или отправьте ему ссылку проекта. Дополнительно можно связаться с другим пользователем по видео связи. Договоритесь заранее с вашим одноклассником о совместной работе.

Самостоятельно ознакомьтесь с бибилиотеками звуков. Выберите "Originals" и составьте трек из трех звуковых дорожек со звуками как на рисунке. Измените темп мелодии до 90 bmp. Следуя рекомендациям прошлых занятий, добавляйте каждый новый звук через каждые 4 такта. Измените громоксть звучания первой звуковой дорожки.

Добавьте 4 звуковую дорожку по вашему усмотрению. Сохраните результат через вкладку меню "File-Save as". Экспортируйте файл в формате mp3 на ваш ПК через "File-Export-Export project to mp3 file".
