Magix music maker

Создание музыкальной композиции с помощью библиотеки звуков

Magix Music Maker - это программа-секвенсор с большим количеством функций для обработки и записи звука. Она находится в условно бесплатном доступе. Без регистрации приложение доступно 7 дней, после прохождения регистрации - 30 дней. Ограниченный период использования подходит для ознакомления и сравнения библиотеки звуков и инструментов с элементами других, бесплатных программ для создания музыки. Преимуществами данного программного обеспечения являются интерфейс на русском языке, а также видео-уроки на официальном веб-сайте. Вы можете просмотреть их по ссылке. Ссылка на скачивание программы также доступна по кнопке.

УРОКИ ПРОГРАММА

Откройте приложение и ознакомьтесь с основными инструментами. В левой верхней части расположены звуковые инструментальные дорожки. Каждая из них отвечает за звучание в музыкальной композиции отдельного инструмента или голоса. Ниже этого блока находятся инструменты для проигрывания, остановки, перемотки всех звуковых дорожек - "Мастер дорожки". На нижнем левом блоке визуализирован музыкальный инструмент, звучание которого определяется бибилиотекой звуков, которую вы выберите. Попробуйте воспроизвести несколько нот. Для удобства клавишами инструмента можно управлять и с помощью клавиатуры. В правой части экрана вы увидите библиотеку инструментов (при приобретении платной версии программы ее можно дополнять), бибилиотеку музыкальных сэмплов. В правой нижней части - список всех звуков инструментов (лупов), вы можете отметить наиболее понравившиеся.

Также для более детального редактирования звука есть инструмент микшер. Его можно найти по вкладке "Вид", далее "Микс", "Микшер".

Блок инструмента "Мастер аудиоэффектов" доступен через путь "Эффекты-Аудио-Блок мастера аудиоэффектов". Он автоматически выставляет уровень частот звуковой дорожки согласно выбранному звукому фильтру. Содержит фильтры как для обработки музыки, так и голоса.

Приступим к созданию собственного музыкального трека. Для этого нам понадобится 5 звуковых дорожек и 5 музыкальных инструментов соотвественно. Для первой дорожки установим инструмент "Percussions", эффект "Right Chime". Перетащите блок на звуковую дорожку. Зажав клавишу Ctrl установите такх 8 блоков в звуковой дорожке. Для дорожек 2-4 выполните те же действия, но для инструментов "Drums" с эффектом "Right Drums N", для "Bass" - "Right Bass C", далее "Guitar"- "Kitty Clean". Оставим 5 звуковую дорожку для инструмента "Fix". Здесь разместим его эффект "Right Bulid Up A" на первые 4 блока (такта). Сюда же добавим  "Rolling Voc FX" на следюущие два такта и "Rolling Reverse" на следюущие два. Звуковая дорожка со всеми инструментами должна иметь вид, как на рисунке. По желанию вы можете продлить звучание какого-либо инструмента, добавив блоки в звуковую дорожку.

Когда композиция готова, прослушайте ее с помощи кнопки "Play". Откройте микшер и запустите трек. Попробуйте изменить громкость кадого отдельного инструмента. Измените общую громкость трека с поощью крайнего правого рычага. 

Для записи трека нажмите на кнопку "Запись звука". Для отмены записи нажмите повторно. Чтобы сохранить трек в формате mp3 передите во вкладку "Файл", далее "Экспорт -  Аудио в формте mp3". Введите название и выберите место хранения файла на вашем ПК. Эта функция доступна только для платной версии приложения.
