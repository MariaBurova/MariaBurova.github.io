РАБОТА С ACOUSTICA MIXCRAFT. Практическая работа. 

Создание первой музыкальной композиции с помощью программы Acoustica Mixcaft

Мы уже в достаточном объеме изучили назначение инструменты аудиоредактора, узнали про библиотеку сэмплов и музыкальных инструментов. Благодаря этой практической работе создадим свою первую композицию.

Цель занятия – создание музыкального фрагмента с помощью секвенсора Acoustica Mixcraft 9. Задачи заключается в освоении практического навыка создания звукового фрагмента, работы со звуковой дорожкой и библиотекой музыкальных инструментов.

Мы приступим к написанию музыкального фрагмента, созданного с помощью имитации звучания фортепиано. Фрагмент будет состоять из повторяющихся звуков, такая форма называется секвенцией.

Первый шаг. Работа с библиотекой звуков. Этот пункт меню находится в инструментах дорожки и называется «Выбор инструмента». Для построения гармоничного звучания мелодии принято использовать, например, звук фортепиано и бас, по желанию можно добавить гитару. Обратите внимание, каким образом проигрываются ноты. Каждую клавишу возможно задействовать с помощью кнопок клавиатуры. 

Можно регулировать силу нажатия на клавишу, настраивать октаву (параметр, отвечающий за высоту ноты), добавлять ноту в видимый диапазон с помощью кнопок «-/+ нота». Предлагается выбрать библиотеку «Synt-Pulsating», в ней – инструменты «Applied Arpeggiator». Воспроизвести пару нот для ознакомления. Для гармоничного звучания следует воспроизвести несколько нот одновременно, демонстрирует одновременное нажатие клавиш «W», «D», «H» из диапазона «С4-F#5».


Теперь перейдем непосредственно к составлению краткого музыкального фрагмента. Предлагается исполнить заранее подобранную партию клавиш: «A», «E», «G», «A», «F», «E», «F», «G», «A». Клавиша «A» зажимается первой, остается в таком состоянии на протяжении игры всей партии, и отпускается последней. Все последующие клавиши поочередно нажимаются и отпускаются примерно каждые 4 секунды. Обучающиеся пробуют воспроизвести партию. Можно экспериментировать, включая одновременно две клавиши в одном такте. 

Следующий этап – запись звуковой дорожки. Как мы уже знаем из прошлого занятия, звуковая дорожка имеет инструмент записи «REC», с помощью него мы ставим метку, какая дорожка сейчас записывается. Для начала записи выбираем инструмент «Запись» в панели управления воспроизведения звука. Сразу после нажатия обучающиеся начинают игру партии. По ее окончании нажимают клавишу «Стоп». Во время записи воспроизводится посторонний звук метронома для соблюдения такта игры партии, он может мешать восприятию звука, но позволяет играть более слаженно. Его звучание можно отключить с помощью меню «Настройки метронома». Обучающиеся после записи воспроизводят звуковую дорожку, чтобы убедиться, что партия сыграна верно, для этого следует нажать клавишу «Проиграть». Если звучание дорожки не устраивает, то ее можно удалить и записать заново.  

Добавляем вторую инструментальную дорожку. Переходя в инструмент «Дорожка»,  выбираем пункт «Инструментальная дорожка». Необходимо выбрать в библиотеке звук «Synth-Pad-Soft», инструмент «Bowed Glass». Для новой звуковой дорожки необходимо записать звучание клавиши «W». Интервал проигрыша можно подобрать самостоятельно. После записи настроить громкость данной дорожки -11 Дб. После того, как вторая дорожка будет успешно записана, перейти к воспроизведению «Мастер-дорожки», это инструмент, который проигрывает одновременно все записанные дорожки. Сохраните результат через меню "Файл".
